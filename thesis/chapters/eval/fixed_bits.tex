\subsection{Evaluation of fixed bits encoding}

In the implementation part of this thesis, I have introduced 3 possible encodings
of input region based on fixed bits, \Cref{enc:fixed_bits,enc:fixed_input,enc:fixed_direct}.
I have meassured computation time of verification task using these encodings
on models M4, M5 and M7 on input regions $R(\text{I0}, \mbox{\{F+1},\ldots,n\})$
and $R(\text{I0}, \{F+1,\ldots,n\})$
where $n$ is the input length of used model, for $F\in \{0, 8, 16, 20\}$.
Results are in~\Cref{tab:eval_input}.
